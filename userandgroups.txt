**1. User Management**

| Command                 | Description                                                                                      |
| ----------------------- | ------------------------------------------------------------------------------------------------ |
| `whoami`                | Displays the current logged-in username.                                                         |
| `id <username>`         | Shows user ID (UID), group ID (GID), and group memberships.                                      |
| `useradd <username>`    | Creates a new user account. <br>Example: `useradd ansible`                                       |
| `passwd <username>`     | Sets or changes the password for a user. <br>Example: `passwd ansible`                           |
| `userdel <username>`    | Deletes a user account (keeps home directory).                                                   |
| `userdel -r <username>` | Deletes a user account **and** removes their home directory. <br>Example: `userdel -r terraform` |
| `su - <username>`       | Switches to another user account (requires password). <br>Example: `su - ansible`                |

---

### üë• **2. Group Management**

| Command                          | Description                                                                               |
| -------------------------------- | ----------------------------------------------------------------------------------------- |
| `groupadd <groupname>`           | Creates a new group. <br>Example: `groupadd devops`                                       |
| `usermod -aG <group> <username>` | Adds an existing user to a supplementary group. <br>Example: `usermod -aG devops ansible` |
| `grep <groupname> /etc/group`    | Verifies group membership. <br>Example: `grep devops /etc/group`                          |
| `cat /etc/group`                 | Displays all existing groups and their members.                                           |

### üóÇÔ∏è **3. System Information Commands**

| Command                                 | Description                               |
| --------------------------------------- | ----------------------------------------- |
| `pwd`                                   | Shows current working directory.          |
| `ls`                                    | Lists files and directories.              |
| `cd /path`                              | Changes directory.                        |
| `cat /etc/passwd`                       | Lists all users registered on the system. |
| `cat /etc/group`                        | Lists all groups.                         |
| `head /etc/passwd` / `tail /etc/passwd` | Displays beginning or end of user list.   |
| `who` / `last`                          | Shows current and previous logins.        |
| `lsof -u <username>`                    | Lists open files by a specific user.      |
